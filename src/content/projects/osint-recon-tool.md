# Passive Recon Intelligence Engine

**Status:** Live
**URL:** [/tools/passive-recon](/tools/passive-recon)

## What It Does

A structured dorking and OSINT checklist built into the portfolio — covering 14 categories of passive reconnaissance techniques across 12+ search engines. Enter a target domain, and the tool generates ready-to-use search queries for Google, Shodan, FOFA, Censys, GreyNoise, GitHub, GitLab, and more.

## Features

- **160+ techniques** across 14 categories (Google Dorking, Shodan, FOFA, Censys, GreyNoise, GitHub Code Search, GitLab, Intelligence X, SSL/TLS Recon, Cloud Asset Discovery, Wayback Machine, Social OSINT, Document Metadata, Bug Bounty Discovery)
- **Target auto-fill** — enter a domain once and all queries update
- **Engine selector** — switch between 12+ search engines with one click
- **Progress tracking** — checkbox system with per-category and overall progress bars
- **Copy & Open** — copy any query or open it directly in the selected engine
- **Filter system** — show only unchecked techniques to focus your workflow
- **Fully client-side** — no data leaves your browser

## Why I Built This

After 1+ year of bug bounty hunting, my recon workflow involved copying dorks from scattered notes and spreadsheets. This tool consolidates everything into a single, systematic checklist I can run through on every new target.
